Learn How to Add HTML Elements

This guide will show you how to add various HTML elements to your web page.

  • Add a Paragraph
  • Add an Image
  • Add a Link
  • Add a Form
  • Add a List

How to Add a Paragraph

To add a paragraph to your web page, use the <p> tag:

<p>This is a paragraph.</p>

This is an example of a paragraph on the webpage. You can add as many paragraphs as you like!

How to Add an Image

To add an image, use the <img> tag with the src attribute:

<img src="image.jpg" alt="Description">

How to Add a Link

To add a clickable link, use the <a> tag with the href attribute:

<a href="https://www.example.com">Click Here</a>

How to Add a Form

To add a simple form with a text input and a submit button, use the following:

<form action="/submit">
    <label for="name">Your Name</label>
    <input type="text" id="name" name="name">
    <button type="submit">Submit</button>
</form>

How to Add a List

To add a list of items, you can create an ordered list (<ol>) or an unordered list (<ul>):

Ordered List (Numbered)

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3

Unordered List (Bulleted)

<ul>
    <li>Item A</li>
    <li>Item B</li>
    <li>Item C</li>
</ul>
  • Item A
  • Item B
  • Item C
